<summary><b>🖐️ HandSign — Real-Time Indian Sign Language Translator</b></summary>
<br/>

Real-time ISL recognition system that tracks hand landmarks via webcam and classifies signs using a custom **6-block Transformer architecture**, enabling two-way sign-to-speech and speech-to-sign communication.

| Attribute | Detail |
|---|---|
| **Stack** | Python, MediaPipe, TensorFlow/Keras, OpenCV |
| **Scale** | Trained on a 156,000-sample synthetic dataset |
| **Performance** | ~87% classification accuracy |
| **Security** | Fully on-device inference, no external data transmission |
| **Impact** | Multi-language, accessibility-first communication tool |
| **Repository** | [ankitjhinkwan/handsign-translator](https://github.com/ankitjhinkwan) |

Designed with a minimal white/black/silver interface to keep the focus on legibility and real-time feedback — built as my flagship demonstration of end-to-end ML system design, from architecture choice through deployment.
